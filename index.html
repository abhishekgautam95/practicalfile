Aim

To write a Python program to calculate the sum of first n natural numbers entered by the user.

Introduction

Natural numbers are the set of positive integers starting from 1 (e.g., 1, 2, 3...). The sum of the first 'n' natural numbers is a fundamental concept in mathematics and computer science. This program demonstrates the use of looping constructs in Python to perform cumulative addition, which is a common pattern in algorithms.

Algorithm

  • Start the program.
  • Input a number 'n' from the user.
  • Initialize sum = 0.
  • Loop from 1 to n. Add loop variable to sum.
  • Display the final sum.
  • Stop.

Source Code

n = int(input("Enter a number: ")) sum = 0 for i in range(1, n + 1): sum = sum + i print("Sum of first", n, "natural numbers is:", sum)

Output

Enter a number: 5 Sum of first 5 natural numbers is: 15

Result

The program successfully calculated the sum of first n natural numbers.

Introduction

The factorial of a non-negative integer 'n', denoted by n!, is the product of all positive integers less than or equal to n. Factorials are widely used in combinatorics, algebra, and mathematical analysis. Example: 4! = 1*2*3*4 = 24.

  • Start.

  • Loop 1 to n. Multiply current value to fact.
  • Print fact.
  • Stop.

Aim

To write a Python program to check whether a given number is a prime number or not.

Introduction

A prime number is a natural number greater than 1 that has only two factors, 1 and itself. Examples include 2, 3, 5, 7, 11, etc. Numbers less than 2 are not prime.

Algorithm

  • Start. Input n.
  • If n <= 1, print Not Prime.
  • Loop i from 2 to n-1.
  • If n is divisible by i, break and print Not Prime.
  • Else print Prime.

Source Code

n = int(input("Num: ")) if n > 1: for i in range(2, n): if (n % i) == 0: print("Not Prime") break else: print("Prime") else: print("Not Prime")

Output

Num: 7 Prime

Result

The number was successfully checked for primality.
